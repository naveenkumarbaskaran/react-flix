# react-flix 🎬

Browse and search movies using The Movie Database (TMDb) API.
Built with React 15, ES6, and Webpack.

![screenshot](https://via.placeholder.com/800x400?text=react-flix)

## Features
- Browse popular movies
- Search by title
- Responsive grid layout
- Loading spinner

## Setup
```bash
npm install
npm start
```
Set your TMDb API key in `.env`:
```
REACT_APP_TMDB_KEY=your_key_here
```

Get a free key at [themoviedb.org](https://www.themoviedb.org/settings/api).
